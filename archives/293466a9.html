еј•иЁҖ

е…ідәҺз”ұCиҪ¬еҗ‘C++зҡ„дёҖдәӣжӢ“еұ•дёҺжіЁж„ҸдәӢйЎ№гҖӮ

дҪҝвҪӨC++еҲ·з®—жі•зҡ„еҘҪеӨ„

  • еңЁе·Із»ҸеӯҰд№ иҝҮCиҜӯвҫ”зҡ„еүҚжҸҗдёӢпјҢеӯҰд№ C++е№¶дҪҝвҪӨе®ғеҲ·з®—жі•зҡ„еӯҰд№ жҲҗжң¬вҫ®еёёдҪҺпҪһеҸӘйңҖиҰҒвјҸдёӘвј©ж—¶е°ұеҸҜд»ҘеӯҰдјҡпҪһ
  • C++еҗ‘дёӢе…је®№CпјҢCиҜӯвҫ”вҫҘвҫҜзҡ„иҜӯжі•вјӨйғЁеҲҶйғҪеҸҜд»ҘеңЁC++вҪӮд»¶дёӯиҝҗвҫҸпјҢжүҖд»ҘеӯҰд№ C++еҜ№еҲ·з®—жі•ж—¶зј–зЁӢиҜӯвҫ”зҡ„иЎЁиҫҫиғҪвј’иҝӣвҫҸжү©е……жңүзӣҠвҪҶе®іпјҢдҫӢеҰӮCиҜӯвҫ”зҡ„иҫ“вјҠиҫ“еҮәпјҲ scanf е’Ң printf пјүвҪҗC++еҝ«пјҢйӮЈд№Ҳе°ұеҸҜд»ҘеңЁдҪҝвҪӨC++еҲ·з®—жі•еҗҢж—¶дҪҝвҪӨ scanf е’Ң printf жҸҗвҫјд»Јз ҒиҝҗвҫҸж•ҲзҺҮ
  • C++жӢҘжңүдё°еҜҢзҡ„STLж ҮеҮҶжЁЎзүҲеә“пјҢиҝҷд№ҹжҳҜPATз”Ізә§гҖҒLeetCodeзӯүйўҳвҪ¬дёӯз»ҸеёёйңҖиҰҒвҪӨеҲ°зҡ„пјҢеҚ•зәҜдҪҝвҪӨCиҜӯвҫ”и§ЈеҶій—®йўҳдјҡвҪҗC++зҡ„STLи§ЈеҶіиҜҘй—®йўҳйә»зғҰеҫҲеӨҡпҪһ
  • C++зҡ„ string и¶…зә§еҘҪвҪӨпҪһвҪҗCиҜӯвҫ”вҫҘвҫҜзҡ„charж•°з»„еҘҪвҪӨеӨҡе•ҰпҪһвҪӨдәҶе°ұеҶҚд№ҹдёҚжғіеӣһеҺ»зҡ„йӮЈз§ҚпҪһ
  • C++еҸҜд»ҘеңЁжҹҗвјҖеҸҳйҮҸдҪҝвҪӨеүҚйҡҸж—¶е®ҡд№үиҜҘеҸҳйҮҸпјҢвҫ®еёёвҪ…дҫҝ
  • еңЁи§ЈеҶівјҖдәӣиҫғдёәз®ҖеҚ•зҡ„PATвј„зә§йўҳвҪ¬зҡ„ж—¶еҖҷпјҲдҫӢеҰӮвјҖдәӣж—¶й—ҙеӨҚжқӮеәҰйҷҗеҲ¶дёҚдёҘж јзҡ„йўҳвҪ¬пјүпјҢ cin гҖҒ cout иҫ“вјҠиҫ“еҮәвҫ®еёёвҪ…дҫҝпҪһвҪӨиҝҮзҡ„йғҪиҜҙеҘҪпҪһ (а№‘вҖў . вҖўа№‘)

иҷҪз„¶C++жҳҜвјҖв»”вҫҜеҗ‘еҜ№иұЎиҜӯвҫ”пјҢдҪҶжҳҜеҜ№дәҺеҲ·з®—жі•иҝҷд»¶дәӢвҪҪвҫ”пјҢжҲ‘д»¬е№¶дёҚйңҖиҰҒжҺҢжҸЎе®ғвҫҜеҗ‘еҜ№иұЎзҡ„йғЁеҲҶпҪһеҸӘйңҖиҰҒжҺҢжҸЎеҲ·з®—жі•зҡ„ж—¶еҖҷйңҖиҰҒвҪӨеҲ°зҡ„йғЁеҲҶпјҲеҹәжң¬иҫ“вјҠиҫ“еҮәгҖҒSTLж ҮеҮҶжЁЎжқҝеә“гҖҒ string еӯ—з¬ҰдёІзӯүпјүе°ұеҸҜд»Ҙе•ҰпҪһCиҜӯвҫ”е’ҢC++жңүеҫҲеӨҡзӣёдјјд№ӢеӨ„пјҢдё”C++еҗ‘дёӢе…је®№CиҜӯвҫ”пјҢжүҖд»ҘжҲ‘жІЎжңүиҜҙзҡ„ең°вҪ…е°ұзӣҙжҺҘвҪӨCиҜӯвҫ”зҡ„иҜӯжі•иЎЁзӨәе°ұеҘҪпҪһд»ҘдёӢжҳҜжӯЈвҪӮпјҢе…ҲжқҘж®өд»Јз ҒвҪ…дҫҝи®Іи§Јпјҡ

еҗҚз§°з©әй—ҙ

иҝҷеҸҘиҜқжҳҜдҪҝвҪӨвҖңstdвҖқиҝҷдёӘеҗҚз§°з©әй—ҙпјҲ namespace пјүзҡ„ж„ҸжҖқпҪһеӣ дёәжңүзҡ„ж—¶еҖҷдёҚеҗҢвјҡе•Ҷе®ҡд№үзҡ„еҮҪж•°еҗҚз§°еҪјжӯӨд№Ӣй—ҙеҸҜиғҪдјҡйҮҚеӨҚпјҢдёәдәҶйҒҝе…ҚеҶІзӘҒпјҢе°ұз»ҷжүҖжңүзҡ„еҮҪж•°йғҪе°ҒиЈ…еңЁеҗ„вҫғзҡ„еҗҚз§°з©әй—ҙвҫҘвҫҜпјҢдҪҝвҪӨиҝҷдёӘеҮҪж•°зҡ„ж—¶еҖҷе°ұеңЁmainеҮҪж•°еүҚвҫҜеҶҷжҳҺвҪӨдәҶд»Җд№ҲеҗҚз§°з©әй—ҙпјҢвјҸд№ҺеңЁC++дёӯдҪҝвҪӨеҲ°зҡ„вјҖдәӣвҪ…жі•еҰӮ cin гҖҒ cout йғҪжҳҜеңЁ std еҗҚз§°з©әй—ҙвҫҘвҫҜзҡ„пјҢжүҖд»ҘеҸҜд»ҘзңӢеҲ° using namespace std;иҝҷеҸҘиҜқвјҸд№ҺжҲҗдәҶжҲ‘жҜҸж®өC++д»Јз Ғзҡ„ж Үй…ҚпјҢе°ұе’Ң return 0; вјҖж ·еҝ…йЎ»жңүпҪһе…¶е®һд№ҹеҸҜд»ҘдёҚеҶҷиҝҷеҸҘиҜқпјҢдҪҶжҳҜдҪҝвҪӨ std вҫҘвҫҜзҡ„вҪ…жі•зҡ„ж—¶еҖҷе°ұдјҡйә»зғҰзӮ№пјҢиҰҒеңЁвҪ…жі•еҗҚеүҚеҠ дёҠ std:: пјҢвҪҗеҰӮеҶҷжҲҗиҝҷж ·пјҡ

жҲ‘и§үеҫ—иҝҷж ·вҪҗиҫғдё‘пјҢжүҖд»ҘдёҚз®ЎиҰҒдёҚиҰҒвҪӨеҲ°пјҢзӣҙжҺҘжҜҸйҒ“йўҳзҡ„д»Јз Ғж Үй…Қеҫ—еҶҷ using namespace std; е°ұеҘҪе•ҰпҪһ

иҫ“е…Ҙиҫ“еҮә

е°ұеҰӮеҗҢ scanf е’Ң printf еңЁ stdio.h еӨҙвҪӮд»¶дёӯвјҖж ·пјҢ cin е’Ң cout еңЁеӨҙвҪӮд»¶ iostream вҫҘвҫҜпјҢзңӢеҗҚеӯ—е°ұзҹҘйҒ“пјҢ io жҳҜиҫ“вјҠиҫ“еҮә input е’Ң output зҡ„вҫёеӯ—вәҹпјҢ stream жҳҜжөҒпјҢжүҖд»ҘиҝҷдёӘ iostream еӨҙвҪӮд»¶вҫҘеҢ…еҗ«зҡ„вҪ…жі•е°ұжҳҜз®ЎзҗҶвјҖдәӣиҫ“вјҠиҫ“еҮәжөҒзҡ„пҪһ

cin е’Ң cout вҪҗиҫғвҪ…дҫҝпјҢдёҚвҪӨеғҸCиҜӯвҫ”вҫҘзҡ„ scanf гҖҒ printf йӮЈж ·еҶҷеҫ—йӮЈж ·з№ҒзҗҗпјҢ cin >> n; е’Ңscanf("%d", &n); вјҖж ·зҡ„ж„ҸжҖқпјҲвҪҪдё”вҪӨ cin еҶҚд№ҹдёҚвҪӨжӢ…вјјеғҸ scanf вјҖж ·еҝҳи®°еҶҷеҸ–ең°еқҖз¬Ұ & дәҶиҖ¶пјүпјҢжіЁж„Ҹ cin жҳҜеҗ‘еҸізҡ„з®ӯеӨҙпјҢиЎЁзӨәе°ҶеҶ…е®№иҫ“вјҠеҲ° n дёӯпҪһ

еҗҢж ·пјҢ cout << n; е’Ң printf("%d", n); вјҖж ·зҡ„ж„ҸжҖқпјҢжӯӨж—¶ cout жҳҜеҗ‘е·Ұзҡ„дёӨдёӘз®ӯеӨҙпјҢжіЁж„Ҹе’Ң cin еҢәеҲҶејҖжқҘпҪһ

вҪҪдё”дёҚз®Ў n жҳҜ double иҝҳжҳҜ int жҲ–иҖ…жҳҜ char зұ»еһӢпјҢеҸӘвҪӨеҶҷ cin >> n; е’Ң cout << n; иҝҷж ·з®ҖеҚ•зҡ„иҜӯеҸҘе°ұеҘҪпјҢдёҚвҪӨеғҸCиҜӯвҫ”дёӯйңҖиҰҒж №жҚ® n зҡ„зұ»еһӢеҜ№еә”ең°еҶҷ %lf гҖҒ %d гҖҒ %c иҝҷж ·йә»зғҰпҪһ

endl е’Ң "\n" зҡ„ж•ҲжһңвјӨиҮҙзӣёеҗҢпјҢ endl зҡ„е…Ёз§°жҳҜend of lineпјҢиЎЁзӨәвјҖвҫҸиҫ“еҮәз»“жқҹпјҢз„¶еҗҺиҫ“еҮәдёӢвјҖвҫҸпҪһпјҲеҫ®вј©еҢәеҲ«жҳҜпјҢдҪҝвҪӨ endl дјҡвҪҗдҪҝвҪӨ "\n" жҚўвҫҸеҗҺеӨҡвјҖдёӘеҲ·ж–°иҫ“еҮәзј“еҶІеҢәзҡ„ж“ҚдҪңпјҢbutдёҚеҝ…careиҝҷдәӣз»ҶиҠӮвҖҰпјүвјҖиҲ¬еҰӮжһңеүҚвҫҜжҳҜдёӘеӯ—з¬ҰдёІеј•еҸ·зҡ„иҜқзӣҙжҺҘ "\n" вҪҗиҫғвҪ…дҫҝпјҢеҰӮжһңжҳҜеҸҳйҮҸд№Ӣзұ»зҡ„жҲ‘и§үеҫ—еҶҷ endl дјҡвҪҗиҫғеҘҪзңӢпҪһжғівҪӨе“ӘдёӘе°ұвҪӨе“ӘдёӘпҪһ

cin е’Ң cout иҷҪз„¶дҪҝвҪӨиө·жқҘжӣҙвҪ…дҫҝпјҢдҪҶжҳҜиҫ“вјҠиҫ“еҮәзҡ„ж•ҲзҺҮдёҚеҰӮ scanf е’Ң printf еҝ«пјҢжүҖд»ҘеҰӮжһңжҳҜеҒҡPATвј„зә§вҫҘвҫҜйӮЈз§Қз®ҖеҚ•гҖҒеҜ№ж—¶й—ҙеӨҚжқӮеәҰиҰҒжұӮдёҚвҫјзҡ„йўҳвҪ¬пјҢзӣҙжҺҘвҪӨ cin е’Ң cout дјҡи§үеҫ—еҶҷиө·жқҘвҪҗиҫғзңҒдәӢвјүпјӣеҰӮжһңйўҳвҪ¬еҜ№ж—¶й—ҙеӨҚжқӮеәҰиҰҒжұӮвҪҗиҫғвҫјпјҢе…ЁйғҪж”№жҲҗ scanf е’Ң printf еҸҜд»ҘжҸҗвҫјд»Јз Ғзҡ„иҫ“вјҠиҫ“еҮәж•ҲзҺҮпјҢвҪҗеҰӮжңүзҡ„ж—¶еҖҷеҸ‘зҺ°вҪӨ cin гҖҒ cout еҒҡйўҳвҪ¬и¶…ж—¶дәҶпјҢж”№жҲҗ scanf е’Ң printf е°ұACдәҶпҪһ

еӨҙж–Үд»¶

C++зҡ„еӨҙвҪӮд»¶вјҖиҲ¬жҳҜжІЎжңүеғҸCиҜӯвҫ”зҡ„ .h иҝҷж ·зҡ„жү©еұ•еҗҺзјҖзҡ„пјҢвјҖиҲ¬жғ…еҶөдёӢCиҜӯвҫ”вҫҘвҫҜзҡ„еӨҙвҪӮд»¶еҺ»жҺү .h з„¶еҗҺеңЁеүҚвҫҜеҠ дёӘ c е°ұеҸҜд»Ҙз»§з»ӯеңЁC++вҪӮд»¶дёӯдҪҝвҪӨCиҜӯвҫ”еӨҙвҪӮд»¶дёӯзҡ„еҮҪж•°е•ҰпҪһвҪҗеҰӮпјҡ

еҸҳйҮҸеЈ°жҳҺ

CиҜӯвҫ”зҡ„еҸҳйҮҸеЈ°жҳҺвјҖиҲ¬йғҪеңЁеҮҪж•°зҡ„ејҖеӨҙпјҢдҪҶжҳҜC++еңЁвҫёж¬ЎдҪҝвҪӨеҸҳйҮҸд№ӢеүҚеЈ°жҳҺеҚіеҸҜпҪһпјҲеҪ“з„¶д№ҹеҸҜд»ҘйғҪж”ҫеңЁеҮҪж•°зҡ„ејҖеӨҙпјүпјҢвҪҪдё”вјҖиҲ¬CиҜӯвҫ”вҫҘвҫҜдјҡеңЁ for еҫӘзҺҜзҡ„еӨ–вҫҜе®ҡд№ү i еҸҳйҮҸпјҢдҪҶжҳҜC++вҫҘвҫҜеҸҜд»ҘеңЁ for еҫӘзҺҜеҶ…йғЁе®ҡд№үпҪһпјҲе…ідәҺиҝҷзӮ№пјҢ VC++6.0 вҫҘвҫҜеҸҜиғҪдјҡеҸ‘зҺ°д»Јз ҒеӨҚеҲ¶иҝӣеҺ»зј–иҜ‘дёҚйҖҡиҝҮпјҢиҝҷжҳҜеӣ дёәиҝҷдёӘзј–иҜ‘еҷЁеӨӘвҪје•ҰпјҢе»әи®®дёҚиҰҒвҪӨиҝҷд№ҲдёҠеҸӨзҡ„зј–иҜ‘еҷЁе•ҰпҪһпјүвҪҪдё”еңЁ for еҫӘзҺҜвҫҘвҫҜе®ҡд№үзҡ„еұҖйғЁеҸҳйҮҸпјҢеңЁеҫӘзҺҜеӨ–вҫҜе°ұеӨұж•Ҳе•ҰпјҲе°ұжҳҜи„ұзҰ»иҝҷдёӘеұҖйғЁдҪңвҪӨеҹҹе°ұдјҡжҹҘвҪҶжӯӨеҸҳйҮҸзҡ„ж„ҸжҖқпјүпјҢжүҖд»ҘвјҖдёӘ main еҮҪж•°вҫҘвҫҜеҸҜд»Ҙе®ҡд№үеҘҪеӨҡж¬ЎеұҖйғЁеҸҳйҮҸ i пјҢеҶҚд№ҹдёҚвҪӨжӢ…вјјеҶҷзҡ„еҫӘзҺҜеӨӘеӨҡеҸҳйҮҸеҗҚ i гҖҒ j гҖҒ k дёҚеӨҹвҪӨе•ҰпҪһ

boolеҸҳйҮҸ

bool еҸҳйҮҸжңүдёӨдёӘеҖјпјҢ false е’Ң true пјҢд»ҘеүҚвҪӨCиҜӯвҫ”зҡ„ж—¶еҖҷйғҪжҳҜвҪӨ int зҡ„ 0 е’Ң 1 иЎЁзӨә false е’Ң trueзҡ„пјҢзҺ°еңЁC++вҫҘвҫҜеј•вјҠдәҶиҝҷдёӘеҸ«еҒҡ bool пјҲеёғе°”пјүзҡ„еҸҳйҮҸпјҢвҪҪдё”C++жҠҠжүҖжңүвҫ®йӣ¶еҖји§ЈйҮҠдёә true пјҢйӣ¶еҖји§ЈйҮҠдёә false пҪһжүҖд»ҘзӣҙжҺҘиөӢеҖјвјҖдёӘж•°еӯ—з»ҷ bool еҸҳйҮҸд№ҹжҳҜеҸҜд»Ҙзҡ„пҪһе®ғдјҡвҫғеҠЁж №жҚ® int еҖјжҳҜдёҚжҳҜйӣ¶жқҘеҶіе®ҡз»ҷ bool еҸҳйҮҸиөӢеҖј true иҝҳжҳҜ false пҪһ

е®ҡд№үеёёйҮҸ

е’ҢCиҜӯвҫ”зӣёеҗҢпјҢC++вҫҘвҫҜеҸҜд»ҘдҪҝвҪӨ const иҝҷдёӘйҷҗе®ҡз¬Ұе®ҡд№үеёёйҮҸпјҢвҪҗеҰӮ int зұ»еһӢзҡ„еёёйҮҸ a иҝҷж ·е®ҡд№үпјҡ

stringзұ»

д»ҘеүҚвҪӨ char[] зҡ„вҪ…ејҸеӨ„зҗҶеӯ—з¬ҰдёІеҫҲз№ҒзҗҗпјҢзҺ°еңЁжңүдәҶ string зұ»пјҢе®ҡд№үгҖҒжӢјжҺҘгҖҒиҫ“еҮәгҖҒеӨ„зҗҶйғҪжӣҙеҠ з®ҖеҚ•е•ҰпҪһдёҚиҝҮ string еҸӘиғҪвҪӨ cin е’Ң cout еӨ„зҗҶпјҢвҪҶжі•вҪӨ scanf е’Ң printf еӨ„зҗҶпјҡ

вҪӨ cin иҜ»вјҠеӯ—з¬ҰдёІзҡ„ж—¶еҖҷпјҢжҳҜд»Ҙз©әж јдёәеҲҶйҡ”з¬Ұзҡ„пјҢеҰӮжһңжғіиҰҒиҜ»вјҠвјҖж•ҙвҫҸзҡ„еӯ—з¬ҰдёІпјҢе°ұйңҖиҰҒвҪӨ getlineпҪһ

sort й»ҳи®ӨжҳҜд»Һвј©еҲ°вјӨжҺ’еҲ—зҡ„пјҢд№ҹеҸҜд»ҘжҢҮе®ҡз¬¬дёүдёӘеҸӮж•° cmp еҮҪж•°пјҢз„¶еҗҺвҫғвј°е®ҡд№үвјҖдёӘ cmp еҮҪж•°жҢҮе®ҡжҺ’еәҸи§„еҲҷпҪһ cmp жңҖеҘҪвҪӨзҡ„иҝҳжҳҜеңЁз»“жһ„дҪ“дёӯпјҢе°Өе…¶жҳҜеҫҲеӨҡжҺ’еәҸзҡ„йўҳвҪ¬пҪһвҪҗеҰӮвјҖдёӘеӯҰвҪЈз»“жһ„дҪ“ stu жңүеӯҰеҸ·е’ҢжҲҗз»©дёӨдёӘеҸҳйҮҸпјҢиҰҒжұӮеҰӮжһңжҲҗз»©дёҚеҗҢе°ұжҢүз…§жҲҗз»©д»ҺвјӨеҲ°вј©жҺ’еҲ—пјҢеҰӮжһңжҲҗз»©зӣёеҗҢе°ұжҢүз…§еӯҰеҸ·д»Һвј©еҲ°вјӨжҺ’еҲ—пјҢйӮЈд№Ҳе°ұеҸҜд»ҘеҶҷвјҖдёӘ cmp ж•°з»„е®һзҺ°иҝҷдёӘзңӢдёҠеҺ»жңүзӮ№еӨҚжқӮзҡ„жҺ’еәҸиҝҮзЁӢпјҡ

жіЁж„Ҹпјҡ sort еҮҪж•°зҡ„ cmp еҝ…йЎ»жҢүз…§и§„е®ҡжқҘеҶҷпјҢеҚіеҝ…йЎ»еҸӘжҳҜ > жҲ–иҖ… < пјҢвҪҗеҰӮпјҡ return a > b; жҲ– иҖ… return a < b; вҪҪдёҚиғҪжҳҜ <= жҲ–иҖ… >= пјҢеӣ дёәеҝ«йҖҹжҺ’еәҸзҡ„жҖқжғідёӯпјҢ cmp еҮҪж•°жҳҜеҪ“з»“жһңдёә false зҡ„ж—¶еҖҷиҝӯд»ЈеҷЁжҢҮй’ҲжҡӮеҒңејҖе§ӢдәӨжҚўдёӨдёӘе…ғзҙ зҡ„дҪҚзҪ®пјҢеҪ“ cmp еҮҪж•° return a <= b ж—¶пјҢиӢҘдёӯй—ҙе…ғзҙ еүҚвҫҜзҡ„е…ғзҙ йғҪвҪҗе®ғвј©пјҢвҪҪеҗҺвҫҜзҡ„е…ғзҙ йғҪи·ҹе®ғзӣёзӯүжҲ–иҖ…вҪҗе®ғвј©пјҢйӮЈд№Ҳ cmp жҒ’иҝ”еӣһ true пјҢиҝӯд»ЈеҷЁжҢҮй’ҲдјҡдёҚж–ӯеҸіз§»еҜјиҮҙзЁӢеәҸи¶Ҡз•ҢпјҢеҸ‘вҪЈж®өй”ҷиҜҜпҪһ

еҲҡеҲҡеңЁеӨҙвҪӮд»¶йӮЈвјҖж®өдёӯд№ҹжҸҗеҲ°пјҢ #include <cctype> жң¬иҙЁдёҠжқҘжәҗдәҺCиҜӯвҫ”ж ҮеҮҶеҮҪж•°еә“дёӯзҡ„еӨҙвҪӮд»¶ #include <ctype.h> пјҢе…¶е®һе№¶дёҚеұһдәҺC++ж–°зү№жҖ§зҡ„иҢғз•ҙпјҢеңЁеҲ·PATвјҖдәӣеӯ—з¬ҰдёІйҖ»иҫ‘йўҳзҡ„ж—¶еҖҷд№ҹз»ҸеёёвҪӨеҲ°пјҢдҪҶжҳҜеҫҲеӨҡвјҲдјјд№ҺдёҚдәҶи§ЈиҝҷдёӘеӨҙвҪӮд»¶дёӯзҡ„еҮҪж•°пјҢжүҖд»ҘеңЁиҝҷвҫҘеҚ•зӢ¬жҸҗвјҖдёӢпҪһ

еҸҜиғҪе№іж—¶жҲ‘д»¬еҲӨж–ӯвјҖдёӘеӯ—з¬ҰжҳҜеҗҰжҳҜеӯ—вәҹпјҢеҸҜиғҪдјҡеҶҷпјҡ

дёҚд»…д»…иғҪеҲӨж–ӯеӯ—вәҹпјҢиҝҳиғҪеҲӨж–ӯж•°еӯ—гҖҒвј©еҶҷеӯ—вәҹгҖҒвјӨеҶҷеӯ—вәҹзӯүпҪһC++е®ҳвҪ…вҪӮжЎЈдёӯеҜ№иҝҷдәӣеҮҪж•°еҪ’зәіжҲҗдәҶвјҖдёӘиЎЁж јпјҢжҲ‘еҲ—еҮәдәҶе®ҳвҪ№зҡ„еҮҪж•°дёҺжүҖеұһиҢғеӣҙжҖ»з»“иЎЁпјҢжңүе…ҙи¶Јзҡ„еҸҜд»ҘзңӢвјҖдёӢпјҡhttps://www.liuchuo.net/archives/2999

C++11

C++11жҳҜ2011е№ҙе®ҳвҪ…дёәC++иҜӯвҫ”еёҰжқҘзҡ„ж–°иҜӯжі•ж–°ж ҮеҮҶпјҢC++11дёәC++иҜӯвҫ”еёҰжқҘдәҶеҫҲеӨҡеҘҪвҪӨзҡ„ж–°зү№жҖ§пјҢвҪҗеҰӮ auto гҖҒ to_string() еҮҪж•°гҖҒ stoi гҖҒ stof гҖҒ unordered_map гҖҒ unordered_set д№Ӣзұ»зҡ„пҪһзҺ°еңЁвјӨеӨҡж•°OJйғҪжҳҜвҪҖжҢҒC++11иҜӯжі•зҡ„пјҢжңүдәӣзј–иҜ‘еҷЁеңЁдҪҝвҪӨзҡ„ж—¶еҖҷйңҖиҰҒиҝӣвҫҸвјҖдәӣи®ҫзҪ®жүҚиғҪдҪҝвҪӨC++11дёӯзҡ„иҜӯжі•пјҢеҗҰеҲҷеҸҜиғҪдјҡеҜјиҮҙзј–иҜ‘еҷЁдёҠзј–иҜ‘дёҚйҖҡиҝҮвҪҶжі•иҝҗвҫҸпјҢжҖ»д№ӢC++11зҡ„иҜӯжі•еңЁOJвҫҘвҫҜжҳҜеҸҜд»ҘдҪҝвҪӨзҡ„пҪһвҪҪдё”еҫҲеӨҡиҜӯжі•еҫҲеҘҪвҪӨпҪһд»ҘдёӢи®Іи§ЈвјҖдәӣC++11вҫҘвҫҜеёёвҪӨзҡ„ж–°зү№жҖ§пҪһ

еҪ“з„¶иҝҷдёӘеңЁз®—жі•вҫҘвҫҜжңҖдё»иҰҒзҡ„вҪӨеӨ„дёҚжҳҜиҝҷдёӘпјҢвҪҪжҳҜеңЁSTLдёӯдҪҝвҪӨиҝӯд»ЈеҷЁзҡ„ж—¶еҖҷпјҢ auto еҸҜд»Ҙд»ЈжӣҝвјҖвјӨв»“дёІзҡ„иҝӯд»ЈеҷЁзұ»еһӢеЈ°жҳҺпјҡ

еҹәдәҺиҢғеӣҙзҡ„forеҫӘзҺҜ

йҷӨдәҶеғҸCиҜӯвҫ”зҡ„forиҜӯеҸҘ for (i = 0; i < arr.size(); i++) иҝҷж ·пјҢC++11ж ҮеҮҶиҝҳдёәC++ж·»еҠ дәҶвјҖз§Қж–°зҡ„ for еҫӘзҺҜвҪ…ејҸпјҢеҸ«еҒҡеҹәдәҺиҢғеӣҙпјҲrange-basedпјүзҡ„forеҫӘзҺҜпјҢиҝҷеңЁйҒҚеҺҶж•°з»„дёӯзҡ„жҜҸвјҖдёӘе…ғзҙ ж—¶дҪҝвҪӨдјҡвҪҗиҫғз®ҖдҫҝпҪһвҪҗеҰӮжғіиҰҒиҫ“еҮәж•°з»„ arr дёӯзҡ„жҜҸвјҖдёӘеҖјпјҢеҸҜд»ҘдҪҝвҪӨеҰӮдёӢзҡ„вҪ…ејҸиҫ“еҮәпјҡ

i еҸҳйҮҸд»Һж•°з»„зҡ„з¬¬вјҖдёӘе…ғзҙ ејҖе§ӢпјҢдёҚж–ӯжү§вҫҸеҫӘзҺҜпјҢ i дҫқж¬ЎиЎЁзӨәж•°з»„дёӯзҡ„жҜҸвјҖдёӘе…ғзҙ пҪһжіЁж„ҸпјҢдҪҝвҪӨ int i зҡ„вҪ…ејҸе®ҡд№үж—¶пјҢиҜҘиҜӯеҸҘеҸӘиғҪвҪӨжқҘиҫ“еҮәж•°з»„дёӯе…ғзҙ зҡ„еҖјпјҢвҪҪдёҚиғҪдҝ®ж”№ж•°з»„дёӯзҡ„е…ғзҙ пјҢеҰӮжһңжғіиҰҒдҝ®ж”№пјҢеҝ…йЎ»дҪҝвҪӨ int &i иҝҷз§Қе®ҡд№үеј•вҪӨеҸҳйҮҸзҡ„вҪ…ејҸпҪһвҪҗеҰӮжғіз»ҷж•°з»„дёӯзҡ„жҜҸвјҖдёӘе…ғзҙ йғҪд№ҳд»Ҙ 2 пјҢеҸҜд»ҘдҪҝвҪӨеҰӮдёӢвҪ…ејҸпјҡ

to_string

to_string зҡ„еӨҙвҪӮд»¶жҳҜ #include <string> пјҢ to_string жңҖеёёвҪӨзҡ„е°ұжҳҜжҠҠвјҖдёӘ int еһӢеҸҳйҮҸжҲ–иҖ…вјҖдёӘж•°еӯ—иҪ¬еҢ–дёә string зұ»еһӢзҡ„еҸҳйҮҸпјҢеҪ“з„¶д№ҹеҸҜд»ҘиҪ¬ double гҖҒ float зӯүзұ»еһӢзҡ„еҸҳйҮҸпјҢиҝҷеңЁеҫҲеӨҡPATеӯ—з¬ҰдёІеӨ„зҗҶзҡ„йўҳвҪ¬дёӯеҫҲжңүвҪӨеӨ„пјҢд»ҘдёӢжҳҜзӨәдҫӢд»Јз Ғпјҡ

  1. дјҡвҫғеҠЁжҲӘеҸ–жңҖеүҚвҫҜзҡ„ж•°еӯ—пјҢзӣҙеҲ°йҒҮеҲ°дёҚжҳҜж•°еӯ—дёәвҪҢ(жүҖд»ҘиҜҙеҰӮжһңжҳҜжө®зӮ№еһӢпјҢдјҡжҲӘеҸ–еүҚвҫҜзҡ„ж•ҙж•°йғЁеҲҶ)

  2. еҰӮжһңжңҖеүҚвҫҜдёҚжҳҜж•°еӯ—пјҢдјҡиҝҗвҫҸж—¶еҸ‘вҪЈй”ҷиҜҜ
